## Installation

使用[`Carthage`](https://github.com/Carthage/Carthage)导入，在项目根目录下创建一个名为`Cartfile`的文件，将下面的类容赋值进去：

```
github "yutingLei/Expander" "master"
```

接下来在终端运行下面命令:

```sh
carthage update --platform iOS
```
